## Git工作流程
0. 進入git所追蹤的資料夾，通常是專案中的頂層資料夾。可以使用`git status`查看。
1. 從遠端版本庫「拉取」資料到本地端（`git pull`）。
2. 更改程式碼。
3. 告訴Git有哪些改變要加到這次的commit中（`git add`）。
4. 執行commit，寫comment說明這次改了什麼（`git commit`）。
5. 將commit後的版本推送到遠端版本庫中，讓其他協作者可以存取（`git push`）。
